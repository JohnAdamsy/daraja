# daraja

A NodeJS library to interact with Safaricom's Daraja M-Pesa API

## Installation

```sh
npm install --save daraja
```

## Usage

Import the DarajaBuilder class as follows:

### JavaScript

```javascript
const {DarajaBuilder} = require('daraja);
```

### TypeScript

```typescript
import { DarajaBuilder } from 'daraja';
```
